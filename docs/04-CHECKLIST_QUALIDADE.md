# Checklist de Qualidade

## Antes de Enviar uma Alteração
- [ ] O teste executa sem falhas locais.
- [ ] O seletor usado é estável e não depende de estrutura frágil.
- [ ] O cenário tem nome claro e objetivo bem definido.
- [ ] Há evidência salva quando o cenário é relevante.
- [ ] As variáveis de ambiente foram verificadas.
- [ ] O relatório foi revisado, se aplicável.

## Revisão de Segurança
- [ ] Nenhuma credencial real foi adicionada ao repositório.
- [ ] Os dados sensíveis continuam fora do controle de versão.
