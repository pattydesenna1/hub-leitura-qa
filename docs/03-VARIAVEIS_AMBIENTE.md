# Variáveis de Ambiente

## Variáveis Utilizadas
- HUB_LEITURA_BASE_URL: URL base da aplicação.
- HUB_LEITURA_USER: e-mail utilizado nos testes.
- HUB_LEITURA_PASSWORD: senha utilizada nos testes.
- HUB_LEITURA_TIMEOUT: tempo máximo de espera em milissegundos.
- HUB_LEITURA_EVIDENCIAS_DIR: diretório para salvar screenshots e evidências.

## Exemplo de Uso
Copie o arquivo .env.example para .env e ajuste os valores conforme o ambiente.

## Observações
Mantenha os dados sensíveis fora do controle de versão. Nunca comite credenciais reais.
